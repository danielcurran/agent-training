# Instructional Design Research Notes

## Initial Questions

- Which instructional design models best translate to agent training?
- How should prompts represent prerequisites, practice, and feedback loops?
- What minimal rubric best captures prompt effectiveness?

## Starter Framework

1. **Define objective**: observable behavior the agent should demonstrate.
2. **Design prompt**: context, constraints, and response format.
3. **Assess output**: score against objective-specific rubric.
4. **Iterate**: refine prompt based on failure patterns.
