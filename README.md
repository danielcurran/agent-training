# Agent Training

A repository for designing, evaluating, and iterating AI agent training materials focused on MongoDB education.

## Repo Structure

```
agents/          → Agent definitions (role, task, behaviors, output format)
labs/
  outlines/      → High-level lab outlines
  specs/         → Detailed technical specs generated by the converter
  reports/       → Evaluation reports generated by the evaluator
standards/       → Instructional design rulebook and research sources
.github/prompts/ → Copilot Chat prompt files (invoke with / in chat)
```

## Agents

| Agent | Purpose | Invoke with |
|---|---|---|
| [Lab Outline Designer](agents/lab-outline-designer.md) | Design new lab outlines | `/design-lab-outline` |
| [Lab Outline Converter](agents/lab-outline-converter.md) | Convert outlines into zero-knowledge technical specs | `/convert-lab-outline` |
| [Lab Instruction Evaluator](agents/agent-lab-evaluator.md) | Evaluate lab specs for training effectiveness | `/evaluate-lab-instructions` |

## Workflow

```
1. Design a lab outline       → /design-lab-outline → labs/outlines/[name]-outline.md
2. Convert to tech spec       → /convert-lab-outline #file:labs/outlines/[name]-outline.md → labs/specs/[name]-tech-spec.md
3. Evaluate the tech spec     → /evaluate-lab-instructions #file:labs/specs/[name]-tech-spec.md → labs/reports/[name]-eval-v1.md
4. Fix issues, re-evaluate    → repeat until score ≥ 8/10
```

## Setup

1. Open this repo in VS Code
2. Ensure `chat.promptFiles` is enabled in settings: `"chat.promptFiles": true`
3. Open Copilot Chat and type `/` to see available prompts
