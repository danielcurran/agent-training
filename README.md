# Agent Training

A repository for designing, evaluating, and iterating AI agent training materials focused on MongoDB education.

## Repo Structure

```
agents/          → Agent definitions (role, task, behaviors, output format)
lab-specs/       → Lab outlines and technical specs
  evaluations/   → Evaluation reports generated by the evaluator agent
prompts/         → Research and reference prompts
.github/prompts/ → Copilot Chat prompt files (invoke with / in chat)
```

## Agents

| Agent | Purpose | Invoke with |
|---|---|---|
| [Lab Instruction Evaluator](agents/ai-agent-training-lab-instruction-evaluator.md) | Evaluate lab specs for training effectiveness + report what MongoDB concepts were learned | `/evaluate-lab-instructions` |
| [Lab Outline Converter](agents/lab-outline-converter.md) | Convert high-level outlines into zero-knowledge technical specs | `/convert-lab-outline` |
| [Instructional Design Coach](agents/prototype-instructional-design-agent.md) | Help design and iterate training prompts | Attach in chat |

## Workflow

```
1. Write a lab outline        → lab-specs/[name].md
2. Convert to tech spec       → /convert-lab-outline #file:lab-specs/[name].md
3. Evaluate the tech spec     → /evaluate-lab-instructions #file:lab-specs/[name]-tech-spec.md
4. Review the evaluation      → lab-specs/evaluations/[name]-tech-spec-eval-v1.md
5. Fix issues, re-evaluate    → repeat until score ≥ 8/10
```

## Setup

1. Open this repo in VS Code
2. Ensure `chat.promptFiles` is enabled in settings: `"chat.promptFiles": true`
3. Open Copilot Chat and type `/` to see available prompts
