# Agent Training Research Repository

This repository is a lightweight workspace for researching how to train AI agents from an instructional design perspective.

## Scope

- Prompt design patterns for guided agent behavior
- Markdown-based research notes and frameworks
- Prototype agent definitions for experimentation

## Repository Structure

- `prompts/` – reusable prompt drafts and templates
- `agents/` – prototype agent specifications
- `docs/` – instructional design notes and research artifacts
