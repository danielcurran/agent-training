# Agent Training

This repository is a lightweight sandbox for agent-training tasks and workflow practice.

## Overview

The project currently contains minimal starter content and is intended to be expanded as tasks are completed.

## Current Repository Contents

- `README.md` – project documentation

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/danielcurran/agent-training.git
   cd agent-training
   ```
2. Create or update project files as needed for your training task.
3. Commit changes on a feature branch and open a pull request.

## Contributing

Contributions are welcome through pull requests. Keep changes focused, document updates clearly, and include any relevant validation steps in your PR description.
